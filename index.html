RaspberryPi 4 CV Security Camera

The "RaspberryPi 4 CV Security Camera" project creates a home security camera system using Raspberry Pi and OpenCV to detect bodies, faces, and objects, triggering video recordings uploaded to Google Cloud Storage and accessible through a React-based frontend on a GCP VM. Additional features include automation triggers based on facial recognition, automated video backups, and integration with Google Cloud SQL for seamless video URL storage and retrieval.

GeoQuizzlr

An engaging trivia application that combines elements of GeoGuessr and Kahoot to test users' geographical knowledge. The app features multiple different types of quizzes and a GeoGuessr-style quiz where users guess countries based on images generated via Google Maps API. Built with React Native, it uses various components and screens for navigation, question handling, and result display. This project demonstrates a robust implementation of interactive quiz features.
